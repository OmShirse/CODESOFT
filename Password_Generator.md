What it does
Generates random passwords so you don't have to use "password123" anymore haha
Features

You choose how long the password should be
Three different strength levels:

Simple (just letters and numbers)
Medium (adds some symbols like !@#$%)
Strong (everything including weird characters)


Can make multiple passwords in one go

How to run it
bashpython password_generator.py
Just answer the questions and it'll spit out a password for you!
Example
How long should the password be? (minimum 4): 12

What kind of password do you want?
1 - Simple (just letters and numbers)
2 - Medium (letters, numbers, and basic symbols)
3 - Strong (everything including special characters)

Pick one (1, 2, or 3): 3

Your generated password is:
----------------------------------------
aK9$mP#2xL@q
----------------------------------------
