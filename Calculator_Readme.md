What it does
Just a simple calculator that can do basic math operations. Nothing fancy but gets the job done!
Features

Addition, subtraction, multiplication, division
Power/exponents (like 2^3)
Square root
Keeps asking if you want to do more calculations
Won't let you divide by zero (learned that one the hard way)

How to run it
bashpython calculator.py
Then just follow the menu! Pick a number for what operation you want, type in your numbers, and boom - you get the answer.
Example
Choose operation:
1 - Addition (+)
2 - Subtraction (-)
...

Pick an option (1-7): 1
Enter first number: 5
Enter second number: 3

5.0 + 3.0 = 8.0
